> # Possíveis classes:

>> - Tarefa
>> - Usuário
>> - TarefaUsuario

> ## Usuario
>> - Criar, editar e excluir usuários
        - Nome
        - Cargo
        - Setor
>> - Mostrar um ou todos os usuários
> ## Tarefa
>> - Criar, editar e excluir tarefas
        - Id
        - Título
        - Descrição
        - Status (
                - espera 
                - análise
                - progresso
                - teste
                - pausada
                - concluída
                - cancelada
        )
        - Prioridade
        - Data criação
        - Data entrega
        - Responsável
>> - Mostrar uma ou todas as tarefas
